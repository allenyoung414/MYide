# iEditor
- 一个Linux下基于Qt 4.8.4 实现的IDE（集成开发环境）
- 纯手写UI代码（没有使用QtCreator下的Design拖控件）
- 跨平台（请看第一、二张图）
- 通过本次开发，让我对C/CC++程序的开发、设计模式、GUI交互、Qt的signal和slot、多线程、进程间通信、跨平台等都有了一个新的认识，是我开启编程之旅上一次成功的历练！
